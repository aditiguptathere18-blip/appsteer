Appsteer 

Project Overview
This is a basic Attendance Management System built using Python Flask.
It includes login functionality and a dashboard.

Technologies Used

- Python
- Flask
- HTML, CSS
- JavaScript (basic)

Setup Instructions

1. Clone the repository

git clone https://github.com/aditiguptathere18-blip/appsteer
cd attendance-system

2. Install dependencies

Make sure Python is installed, then run:
pip install flask

3. Run the application

python app.py

4. Open in browser
Go to:
http://127.0.0.1:5000/
